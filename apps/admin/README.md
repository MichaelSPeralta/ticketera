# 🖥️ Ticketera Admin Dashboard

This is the management interface for the Ticketera system. It allows support agents to view, manage, and resolve tickets.

## Features
- **Ticket List**: View all tickets from your organization.
- **Context Viewer**: See the full conversation history that led to the ticket creation.
- **Priority Management**: Filter and sort tickets by urgency.

## Getting Started

1. Ensure the API is running (usually on port 3001).
2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Technologies
- **Next.js 15** (App Router)
- **React 19**
- **Tailwind CSS 4**
- **Shadcn/UI**
- **Lucide Icons**
